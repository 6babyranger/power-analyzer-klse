# POWER Analyzer 🚀

AI-powered KLSE stock analyzer dengan real-time price alerts, portfolio tracking, dan 15-point scoring system.

## Features

✅ **20 Best Stocks Auto-Scan** - Auto-load setiap app buka
✅ **15-Point Scoring** - Score 0-100 based on momentum, volume, technical
✅ **Entry / TP / SL** - Harga masuk, take profit, stop loss
✅ **Favorites** - Star saham yang kau beli untuk monitor
✅ **Price Alerts** - Notify kalau stock hit Entry/TP/SL
✅ **Buy History** - Track semua trade dengan P&L real-time
✅ **Live Price Updates** - Update harga setiap buka app

## How to Use

1. Open artifact dalam Claude.ai
2. App auto-load 20 best stocks
3. Star favorites untuk monitor
4. Set price alerts untuk Entry/TP/SL
5. Record buy transactions
6. Track portfolio profit/loss

## Trading Tips

- Start dengan RM 1,000 capital
- Follow Entry price & TP exactly
- Use Stop Loss ALWAYS
- Target 5-10% return per trade
- Reinvest profit untuk scale
